
# 15日天气预报查询系统(15d weather forcaster)

## 功能简述

本项目主要实现的功能是根据输入的省份名称，以 html 中可拖动的轮播图形式展示该省未来 15 天的温度预报，并且可查看该省份各城市的温度细节。还可以根据需要修改显示最高温度或最低温度分布，默认显示最高温度分布。

## 运行方式

双击run.bat即可运行程序，或者在python环境下运行main.py即可运行。
